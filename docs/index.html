Readable ffprobe
for the terminal

Print video file details in labeled sections instead of a raw ffprobe dump. Works on one file or a whole folder. Compare tables, compact lines, or JSON you can pipe to jq.

Install

One binary. Put ffprobe (from FFmpeg) on your PATH.

From source

git clone https://github.com/Alyetama/vidinfo
cd vidinfo
cargo install --path .

ffprobe

# macOS
brew install ffmpeg

# Debian / Ubuntu
sudo apt install ffmpeg

Usage

Pass files, folders, or globs. If one file fails in a batch, the others still run.

One file vidinfo clip.mp4
Folder summary vidinfo --compact ./clips/
Compare vidinfo a.mp4 b.mp4 c.mkv
JSON to jq vidinfo --json clip.mp4 | jq .
Field filter vidinfo --fields codec,resolution clip.mp4
Recursive vidinfo -r --sort-by size ~/Movies/

Flags

Not many options. The defaults are usually enough.

FlagDescription
PATHS…Files, directories, or globs
-r, --recursiveWalk subdirectories
-c, --compactOne line per file
--compareSide-by-side comparison table
--sort-byname, duration, size, resolution, bitrate, or codec
--jsonNormalized structured output
--fieldsComma-separated sections or fields
--color / --no-colorForce ANSI on or off

Exit 0 ok · 1 all failed · 2 partial · 127 ffprobe missing

Out of scope

Read-only. No playback, thumbnails, waveforms, or transcoding.
